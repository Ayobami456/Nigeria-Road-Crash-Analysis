Nigeria Road Traffic Crash Analysis (2020–2024)

Overview



This project analyzes road traffic crashes across all Nigerian states from 2020 Q4 to 2024 Q1, uncovering trends in crashes, fatalities, injuries, and contributing factors. The dashboard was designed to allow interactive exploration of trends and risk factors across states and time periods, providing actionable insights for policymakers, traffic safety analysts, and researchers.



Tools \& Technologies



Python (Pandas, Matplotlib) – data cleaning, transformation, visualizations



Power BI – interactive dashboard with filters and custom tooltips



Highlights



Developed an interactive dashboard to explore crash trends by state, year, and contributing factors



Calculated key metrics including Fatality Rate, Injury Rate, and average Vehicles per Crash per state



Identified high-risk states and major crash contributors such as speed violations and fatigue



Designed professional, visually appealing dashboards with consistent color palette and clear storytelling



Impact



Provides actionable insights for policymakers and traffic safety analysts



Allows stakeholders to quickly identify trends and prioritize interventions



Demonstrates expertise in Python data analysis and interactive Power BI reporting
